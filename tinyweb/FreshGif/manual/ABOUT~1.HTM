About

Introduction

GIFLIB is an easy to use library that enables you to use the GIF file format in your programs. It comes as a static library which can be linked to your program. GIFLIB is very small. This documentation is focussed on win32asm programming, if you want to use it with other languages, you'll have to convert the include files. GIFLIB is focussed on MASM users.

Specifications

Supported

  • 2, 4, 8, 16, 32, 64, 128 and 256 color gifs
  • Outputting data as 1-bit, 4-bit and 8-bit data
  • Global or local color tables
  • Exporting the color table to RGBQUAD format
  • Useful functions for implementing with the windows GDI

Not (yet) supported

  • Interlaced gifs
  • Non-image drawing extensions (plain text extension)
  • Graphic control extensions (transparency, etc)
  • All other things not mentioned under the supported section ;-)

More

  • GIFLIB is entirely written in assembly language (win32asm)
  • GIFLIB is very small (around 5kb)
  • Your programs will grow just about 2kb when using the library
  • GIFLIB is free!
